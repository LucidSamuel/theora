# Contributing to Theora

Found a bug or have an idea? Open an issue.

**Bug reports**: Include the Share URL (click "Copy Share URL" in the demo sidebar) so we can reproduce the exact state. Screenshots help too.

**Feature requests**: Describe what you'd like to see and why. If it's a new cryptographic primitive, mention the paper or construction.

**Pull requests**: Fork the repo, make your changes, and open a PR against `main`. Run `npm run verify` locally before submitting — it runs tests, type-checks, and builds.
